Built for repeated use

A quiet desktop utility for everyday writing.

Correct anywhere

Use a global shortcut from Mail, Slack, Chrome, notes, or any app where you can select text.

Review before applying

Compare original and corrected text side by side, then accept, copy, refine, or reject the result.

Custom styles

Proofread, rewrite professionally, shorten, elaborate, translate, or create your own reusable style.

SnapApply shortcuts

Assign a hotkey to a style and apply common corrections in one step without opening the popup.

Gemini or offline

Use Gemini with your own verified API key, or run downloaded Gemma 4 GGUF models locally.

Local-first settings

Manage models, permissions, styles, shortcuts, startup behavior, and language from the desktop app.

Privacy by default

Private offline mode. Content-free analytics.

Offline models run on your device.

Gemini is optional and uses your own API key.

Anonymous analytics are opt-in and never include selected text, prompts, outputs, images, filenames, API keys, or raw errors.
